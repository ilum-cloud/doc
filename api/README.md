# Ilum API Changelog

### 1.2.0
- Add /job/{jobId}/logs endpoint

### 1.1.0
- Enhanced /group and /job endpoints' GET methods with the clusterNames filtering parameter for improved querying.
- Introduced the /cluster/names endpoint, featuring a GET method that returns a list of names for all existing clusters.

### 1.0.0
- Established the OpenAPI schema for the Ilum platform.
